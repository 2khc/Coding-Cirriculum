Chapter 6: Arrays

C provides a capability that enables you to define a set of ordered data items known as an array. This chapter will teach you how arrays can be used and manipulated.

What is an Array?

An array is a type of data-structure that holds an ordered set of data all of the same type. The size of the array is predetermined, so if one were to declare an array of size 5, the maximum number of values it could ever hold would be 5, and no greater.

Defining an Array

To name an array the same rules you use for naming a variable are applied. At the end of an array’s name, the number inside a pair of square brackets implies the capacity of that particular array. For example, “int people[4]” contains the maximum number of four data items.

Initialising Arrays

The way you initialise values to an array is pretty similar to how you do it with variables except that you only assign one value to a variable.

For example:

 int main(void){
   int people[4] = {2, 4, 83, 9};
 }

Get an access to arrays

Array[n] is indexed from 0 to n-1. So you can get an access to a particular index using array[n] given that n is the index number whose stored valued is required.

Exercises

  1. Define an array of size 5, holding the following elements: 2 15 32 37 48
  2. Define another array of size 7, holding the following elements: 1, 13, 34, 56, 78, 85, 99
  3. Access the fourth element in the first array, and the fifth element in the second array
  4. Have your compiler add these two elements together, without explicitly writing the numeric values (your input should look something like myArray[x] + mySecondArray[y])
  5. If your answer wasn't 115, then you need to go over array indexing! Remember we start counting at 0, not 1!
  6. Assign the values 51 and 66 to the fourth and fifth elements of each array respectively.
  7. Again, add them together. Implement your revision of array indexing from questions 4 and 5

Then, have your program add together the products of each pair of corresponding numbers. However, there is a catch! You cannot do this manually, by typing out all the maths yourself.
