Conclusion

We've finially covered all we need to know in order for you to create a simple C programme. Of course there's a lot more to programming but we believed that to make our couse as precise as possible is very important. This course has always been about introducing you to programming so making things seem easy was one of our main objectives. In other words, we did not want to disencourage you from learning to code in any way.

I hope that you had a good time learning through our course and this had made you want to learn more about coding. If that's the case then I highly recommend you to get yourself a proper programming textbook on C. The one I and many others find helpfull is called "Programming in C" by Stephen G. Kochan.

Last but not least, I wish you all the best wheather our course made you want to pursue a degree in Computer Science or not and hope to see you real soon!
