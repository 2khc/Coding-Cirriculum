Programming Basics

Before we can get into any coding stuff, we first need to understand some basic rules of programming first. Programming is pretty much all about taking in inputs and giving you out desired outputs. To make things a little bit more organised and understandable, these inputs and outputs are categorised into different groups called data types and are usually stored in variables.

What is that program written in the 1st chapter?

Here is the short program that you wrote and ran in the last chapter:

 #include <stdio.h>

 int main(void)
 {
     printf("Hello, world");
 }

The first line of that program is '#include ', what this does is imports code into your program so that you can use things that have already been written by other programmers for you to use. We programmers don't like to reinvent the wheel. The library being imported is 'stdio', which stands for standard input output. It allows use to use the printf function. The thing we put in-between the parentheses is called an argument to the function. It is the information which we input into the function for it to create some output. The output in this case is 'Hello, world'.

The next thing to consider is the 'int main(void)'. What is this? We can see that int main(void) is followed by a curly bracket '{'. These curly brackets always come in pairs. You can see that the one that completes the pair is the '}' at the very end of the program. Everything between these 2 curly brackets belong in the 'main' function.

Another important thing to remember is that the main function is the beginning of every C program.

 int main(void)
 {

 }

This has to exist in every C program and it is where everything begins.

Data Types

The C programming language provides five basic types: int, float, double, char and boolean

The Integer Type int

A variable declared to be of type int can only contain integer values or values that do not have decimal places (and are therefore whole numbers). To display an int value at the terminal, the printf conversion characters %iV are used.

How would one declare a variable of type int?

 int myVariable;

It's as simple as that! If you want, you can also add a value to it as you're intialising it like so:

 int myVariable = 5;

Examples: -23, -83, 0, 435 and 1

The Floating Number Type float

A variable declared to be of type float can be used for storing values containing decimal places. To display an int value at the terminal, the printf conversion characters %f are used. To express a value of type float, f or F is put after a value such as 14.5f or -122.23F.

Examples: 3.0, 23.42, -0.3329 and 9.023

Similar to the integer declaration, a float declaration would be as follows:

 float myVariable;

The Type double

The double type is very similar to the float type except that it is more precise. It can store approximately as many significant digits as can a variable declared to be of type float. All floating-point values are taken as the double type unless told otherwise.

 double myVariable;

The char type

The char type is used to store a single character. Characters can be declared by enclosing a pair of quotation marks around them as follows; ‘a’, ‘D’, ‘3’ and ’T’. To display a char value at the terminal, the printf conversion characters %c are used

 char myVariable;

The Boolean Data Type boolean

A variable declared to be of type boolean can only store 1 and 0 or True or False. boolean is used to indicate a Boolean condition or a condition which can be outputted to either true or false.

Variables

Variables are named and used to store program computations or results. Variables can be named in a way that reflect the usage of the data stored. For example, I can create a variable called “numberOfpens” to store an int type which represents the number of pens I possess.

numberOfpens = 4;

Naming Conventions

There are certain rules that we need to follow when it comes down to naming variables.

Variable names can’t begin with a number. Embedded spaces are not permitted. Variable names can’t be those that are same to reserved words in C e.g. int, float and so on. $ is not a valid character.

Although we cannot include any spaces in our variable names, programmers tend to use underscores( _ ) or uppercase letters to represent next words.

 calculate_angle
 isTrue

Arithmetic Expressions

C’s arithmetic expressions work pretty much the same way as those we use in the real-world. However, some of the maths rules might not be as you expect depending on the type that a variable is declared to be.

When there are two or more operations the idea of priority or precedence is introduced to determine which operation is to be evaluated first.

The following notions are listed in an order of precedence.

multiplication sign -> ( * ) division sign -> ( / ) plus sign -> ( + ) minus sign -> ( - )

A Few Extra Operations to consider

When you have an 'integer' variable, you can increment its value as follows:

 int number = 1;
 number = number + 1;

This would cause the value of number to increment by 1, as say the new value of number is the current value plus 1. This is often represented in short-form like so:

 number++;

This is basically identical to 'number = number + 1;'. How about if you wish to increment by more than 1?

 number += 2;

Using '+=', we can cause an increment of +2, or whatever value we choose.

Finally, all all these work with decrementing values if we replace the '+' with a '-', pretty neat?

 number--;
 number -= 2;

Programmers like to be lazy, this is just one of the many ways they chose to be so.

Exercises: Let's Write a Basic Calculator!

The first thing we need to do is prepare to prepare the file for use. We are going to be displaying things on the terminal, and asking for input, so we'll need to include the 'stdio' library.

 #include <stdio.h>

 int main(void)
 {

 }

We now need to ask for user input, and store the input into some variables, so that we can add them together later.

#include <stdio.h>

int main(void)
{
  int numberOne;
  int numberTwo;

  printf("Enter 2 numbers to add together:\n");
  scanf("%d", &numberOne);
  scanf("%d". &numberTwo);

  printf("%d + %d = %d", numberOne, numberTwo, numberOne + numberTwo);
}

Some new things have creeped up in this exercise. What does this 'scanf' do? Well, it takes input from the console then puts it into the variable stated in the second argument of the function.

  scanf("%d", &numberOne);

This takes your input value and assigns it to numberOne, such that numberOne holds whatever whole number you decide to input.

But wait a second, what is this "%d" that is in both the 'printf' and 'scanf' in the first arguments? This "%d" is a formatter that is applied to the value of the argument in the same function. For example, in the above example of scanf, the number input into the terminal is then assigned to the variable 'numberOne'.

In the printf, 'numberOne' is placed into the first %d, 'numberTwo' is placed into the second %d, the value of the sum of 'numberOne + numberTwo' is placed into the 3rd %d.
